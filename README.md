# Simple RSS Aggregator with auto-update

### Demo
### http://rss-aggregator.surge.sh/

---

### 1. Install dependencies:

```
$ make install
```

### 2. Run CORS-proxy:

```
$ make cors
```
### 3. Run dev-server:

```
$ make dev
```
### 4. Build project:

```
$ make build
```
